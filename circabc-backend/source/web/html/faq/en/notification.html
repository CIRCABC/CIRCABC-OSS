  • What are notifications?

    Notifications are messages sent automatically by CIRCABC to let you know that a new document has been uploaded to the library or a new topic has been posted to the newsgroup.

    Top of the page

  • How to subscribe to notifications

    By default notifications are disabled, CIRCABC does not send any notifications to the new invited users.

    To receive notifications click on Administration (from the Interest Group homepage; from any other page of the group, follow this path: Administration > Go to the interest group administration page)

    Click on My Notifications Status
    Then Enable the Global Notification Status and Subscribe to the Notification Status

    Top of the page

  • How can Leaders enable notifications for their members?

    On Administration (from the Interest Group homepage; from any other page of the group, follow this path: Administration > Go to the interest group administration page)

    Click on Notification Status and Define Notification Status

    Leaders can enable notification per user or per profile (group of users)

    Select the user or profile (to subscribe all members: select "Profile", keep the box empty and click on Search)
    Select the user or profile from the list (to subscribe all members: select all profiles)
    Assign a Notification Status: Subscribed
    Click on Add to List
    Then OK

  • I am not receiving notifications, what can I do?

    If you are not receiving notifications, please check the following points:

    By default notifications are disabled, CIRCABC does not send any notification to the new invited users.

    - Check the Global Notification:

    Log into CIRCABC, Click on Administration, My account,
    In User Options check if the Global Notification is enabled

    - From the home page of your Interest Group: check if your notifications are enabled

    How to subscribe to notifications

    - From the library service: check the notifications status:

    From the library click on Administration, My Notification status
    Global notification should be enabled
    Notification Status should be Subscribed or Inherited if you have subscribed from the Interest group level.

    Top of the page

  • How to stop receiving notifications

    Log into CIRCABC, Click on Administration, My account,
    In User Options, disable the Global Notification.
