Dutch Language Pack - Nederlandse vertaling

Installatie

Zie ook de Alfresco Wiki: http://wiki.alfresco.com/wiki/Language_Pack_Installation . De *nl.properties bestanden zijn de algemeen Nederlandse bestanden. De *nl_BE* verbijzonderd naar Nederlands-België. De *nl_NL* verbijzonderd naar Nederlands-Nederland.

Voor Tomcat

*nl*.properties - kopiëren naar <Tomcat home>/shared/classes/alfresco/messages

Voor JBoss

*nl*.properties - kopiëren naar <JBoss home>/server/default/conf/alfresco/messages

Login Language Options

To change the language options offered on the login screen, you need to modify the web-client-config-custom file (you may need to rename the sample file):

  • Edit 'web-client-config-custom.xml' file in the <custom config>/extension folder
  • Find the '<languages>' section
  • Add or remove language entries of the form '<language locale="XX_YY">LangName</language>'
  • Save the file 

 <config evaluator="string-compare" condition="Languages">
 <languages>
 <language locale="nl_BE">Nederlands (België)</language>
 <language locale="nl_NL">Nederlands (Nederland)</language>
 </languages>
 </config>

Changelog - Geschiedenis

2008 06 05 - Alfresco 2.9B nl 04: Ongeveer 80% van de teksten is vertaald. De *nl_BE en *nl_NL zijn leeggemaakt.

2008 05 25 - Alfresco 2.9B nl 03: Ongeveer 75% van de teksten is vertaald. Op veel plaatsen is de vertaling consequenter gemaakt.

2008 05 18 - Alfresco 2.9B nl 02: Ongeveer 60% van de teksten is vertaald. De bestaande vertaling is verbeterd door consequent gebruik van dezelfde vertaaltermen voor woorden met dezelfde betekenis. Op veel plaatsen is de gebiedende wijs vervangen door de onbepaalde wijs. Voorbeelden: "Gebruikers beheren" in plaats van "Beheer gebruikers", "Knippen" in plaats van "Knip". Het pakket is nu gestructureerd volgens:
*nl.properties - algemeen Nederlandse vertaling, zowel Vlaams als Nederlands
*nl_BE - Vlaams, Nederlands-België
*nl_NL - Nederlands-Nederland
In deze versie zijn de *nl_BE* bestanden van de vorige vertaling (Alfresco 1.3). Door dus te kiezen voor de nl_BE vertaling heeft u nog de beschikking over de vorige teksten aangevuld met nieuwe.

2008 05 16 - Alfresco 2.9B nl_NL 01: De eerste beta release van de Nederlandse vertaling voor Alfresco 2.9B. Vertalingen voor Alfresco 1.3, 1.4 en 2.0 zijn samengevoegd en deels aangepast. De teksten in de nl_NL-01 release zijn voor 46% vertaald. De kwaliteit van de vertalingens is nog niet op 100%.
